About

This document is automatically generated from the station correlation reports via a series of python scripts. As such, please alert the creators of any apparent issues at the following address tiegem@utas.edu.au.

Station position figures are based on analysis output from the Paris Observatory Geodetic VLBI Center.

This report was developed by the AuScope VLBI project, which is managed by the University of Tasmania, contracted through Geoscience Australia.

More information on this project as well as the source code may be found at the following link.

Notes

1 This value represents the stations contribution to the final session fit, it is extracted directly from the IVS analysis reports.

2 The performance values represent the fraction of observations used in analysis compared to scheduled. These values are extracted directly from the IVS analysis reports. Performance values are heavily affected by other station outages, and are not necessarily indicative of issues at the station.

3 Assignment rate is calculated by dividing the number of observations assigned to a station by the maximum observations assigned to any station in that session. The total observations for each station per session are extracted from the correlation reports. Please note that assignment rates will be much higher for stations in denser parts of the network.

4 Detection rates are calculated by dividing the number of observations with Q codes 6 or above by the total number of observations. This value is meant to give an indication of the sensitivity of baselines this station participates in.
